**Durable** runs long-lived, resumable workflows across apps and restarts. Steps are checkpointed to a store and driven over a transport, so a workflow survives crashes, deploys and retries — like an albatross that never has to land.

<Callout title="Aviary call sign: Albatross">
Part of the **Aviary** ecosystem — every library reads from a shared [Context](/docs/context),
so adopting one makes the next one cheaper.
</Callout>

## Install

```bash
pnpm add @dudousxd/nestjs-durable
```

## Two ways to fly

### Plug-n-play

Define a workflow, pick a store and a transport, and run it. Built-in dashboard, CLI and OpenTelemetry give you visibility from minute one.

### Fully configurable

Mix and match: Prisma / Drizzle / TypeORM / MikroORM stores, BullMQ / SQS / DB / event-emitter transports, plus codegen, an ESLint plugin and a Telescope bridge.
