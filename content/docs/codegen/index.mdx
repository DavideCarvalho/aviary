**Codegen** weaves a fully-typed client out of your NestJS app. Pick the validation library you already use — Zod, Valibot or ArkType — and optionally emit TanStack Query hooks and superjson transport. Works with or without Inertia.

<Callout title="Aviary call sign: Weaver">
Part of the **Aviary** ecosystem — every library reads from a shared [Context](/docs/context),
so adopting one makes the next one cheaper.
</Callout>

## Install

```bash
pnpm add @dudousxd/nestjs-codegen
```

## Two ways to fly

### Plug-n-play

Point codegen at your app, choose a validator preset, and get a ready-to-import typed client. One command, sane defaults.

### Fully configurable

Every emitter is a plugin: swap the validator, add TanStack Query, change the transport, or write your own artifact. The pipeline is the public API.
