**Telescope** is an observability console for NestJS in the spirit of Laravel Telescope. Watchers record requests, queries, jobs, cache, mail, events and more — correlated per request — behind a headless API with an optional dashboard UI.

<Callout title="Aviary call sign: Owl">
Part of the **Aviary** ecosystem — every library reads from a shared [Context](/docs/context),
so adopting one makes the next one cheaper.
</Callout>

## Install

```bash
pnpm add @dudousxd/nestjs-telescope
```

## Two ways to fly

### Plug-n-play

Register the module and open the dashboard. Requests, exceptions and queries start showing up immediately, correlated by traceId.

### Fully configurable

Enable only the watchers you want, choose pluggable storage (Prisma / TypeORM / MikroORM / Redis), add OTel export, an AI watcher, and bridges for Durable and Inertia.
