**Inertia** is a first-class Inertia.js adapter for NestJS. Render client pages straight from controllers, run multiple Inertia apps side by side, and call your backend through a Tuyau-style fully-typed client. Vite integration is built in.

<Callout title="Aviary call sign: Swallow">
Part of the **Aviary** ecosystem — every library reads from a shared [Context](/docs/context),
so adopting one makes the next one cheaper.
</Callout>

## Install

```bash
pnpm add @dudousxd/nestjs-inertia
```

## Two ways to fly

### Plug-n-play

Return `inertia.render('Page', props)` from a controller and you're rendering a SPA page — SSR, asset versioning and shared props handled.

### Fully configurable

Wire multiple apps, customise the Vite pipeline, generate the typed client, and use the testing utilities for end-to-end coverage.
