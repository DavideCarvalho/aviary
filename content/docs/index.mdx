**Aviary** is a collection of libraries for NestJS that are designed to live together —
in the spirit of [TanStack](https://tanstack.com) for the React world and
[Laravel](https://laravel.com) for the PHP world. Every library is independent, every
library is typed end-to-end, and every library ships under the `@dudousxd` scope.

Two promises hold across the whole habitat:

1. **Plug-n-play** — import a module, get defaults that work. No config marathon.
2. **Fully configurable** — every adapter, store, transport and validator is a public API.

They also share a common nervous system: [Context](/docs/context). Adopt one library and
the next one is cheaper, because they already speak the same language.

## Conventions

Across every package you'll find the same shapes:

| You'll see | It means |
| --- | --- |
| `XxxModule.forRoot(options)` | Global, configured-once setup. |
| `XxxModule.forFeature(...)` | Per-feature / per-module registration. |
| `@dudousxd/nestjs-<lib>-<adapter>` | An optional adapter you opt into. |
| `@dudousxd/nestjs-<lib>-testing` | First-class testing helpers. |

Ready? Open a specimen from the sidebar, or jump straight to
[Context](/docs/context) — the library everything else builds on.
